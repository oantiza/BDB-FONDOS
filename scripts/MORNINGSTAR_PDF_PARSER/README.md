# Compatibility Wrappers

El parser Morningstar PDF operativo vive ahora en:

`MORNINGSTAR_PDF_PARSER/`

Esta carpeta bajo `scripts/` queda solo por compatibilidad con rutas antiguas.

Usar preferentemente:

```bash
node MORNINGSTAR_PDF_PARSER/bin/parse_dry_run.js
```

No colocar PDFs ni resultados aqui.
