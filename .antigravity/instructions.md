🧩 REGLAS DE ORO DEL PROYECTO (SYSTEM PROMPT)
🛠 STACK TECNOLÓGICO OBLIGATORIO
Framework: React / Next.js (App Router).

Estilizado: Tailwind CSS (Estilo Premium: minimalismo, espaciado amplio, tipografía Sans-Serif moderna).

Backend/DB: Firebase Suite (Firestore, Auth, Storage).

State Management: Hooks nativos o Zustand (evitar Redux si es posible).

🧠 COMPORTAMIENTO DEL AGENTE
Eficiencia de Tokens: Sé extremadamente conciso. Si el código habla por sí solo, no expliques cada línea.

Modo Supervivencia (Flash): Si detectas que la cuota del modelo Pro es baja o estás operando en Gemini 3 Flash, duplica la verificación de sintaxis para evitar alucinaciones.

Arquitectura de Firebase: No preguntes por la estructura de la DB. Usa siempre patrones de documentos planos y reglas de seguridad de "denegar por defecto".

Strict Mode: Respeta siempre las restricciones de seguridad y no sugieras librerías externas pesadas sin consultar.

🎨 UI/UX GUIDELINES
Interfaz limpia, bordes redondeados (rounded-xl), sombras suaves y modo oscuro como prioridad.
